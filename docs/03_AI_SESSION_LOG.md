# Purpose

Este documento registra las sesiones de trabajo realizadas con inteligencia artificial durante el desarrollo de **Narrative Engineering**.

Su objetivo es conservar la trazabilidad del proceso creativo sin necesidad de registrar las conversaciones completas.

Cada sesión documenta el contexto de trabajo, los temas tratados y los principales resultados obtenidos.

Este documento responde únicamente a la siguiente pregunta:

> **¿Qué se discutió durante el desarrollo del proyecto?**

---

# Scope

El AI Session Log documenta el proceso de trabajo.

No documenta:

- decisiones oficiales del proyecto
- reglas metodológicas
- canon del universo
- cambios del repositorio

Su propósito es preservar la memoria del proceso creativo.

---

# Session Principles

Cada sesión deberá resumirse de forma objetiva.

El registro deberá describir:

- qué se discutió
- qué conclusiones se alcanzaron
- qué documentos fueron creados o modificados
- qué próximos pasos quedaron definidos

No es necesario registrar la conversación completa.

---

# Session Template

Cada sesión utilizará la siguiente estructura.

```markdown
## Session XXX

**Date**

YYYY-MM-DD

**Author**

Nombre del autor.

**AI**

Nombre de la inteligencia artificial.

**Model**

Modelo utilizado.

**Objective**

Objetivo principal de la sesión.

**Topics**

- Tema 1
- Tema 2
- Tema 3

**Summary**

Resumen de la conversación.

**Outcome**

Resultado obtenido durante la sesión.

**Documents**

- Documento creado.
- Documento actualizado.

**Next Steps**

- Pendiente 1.
- Pendiente 2.
```

---

# Sessions

Las sesiones se documentarán en orden cronológico.

Cada sesión deberá utilizar un identificador consecutivo.

Ejemplo:

- Session 001
- Session 002
- Session 003

---

# Documentation Principles

Cada sesión deberá ser:

- breve
- objetiva
- clara
- fácil de consultar

El objetivo no es reemplazar la conversación original.

El objetivo es conservar la evolución del pensamiento del proyecto.

---

# Final Statement

Narrative Engineering considera que el proceso creativo también forma parte del conocimiento del proyecto.

Las sesiones con inteligencia artificial permiten comprender cómo evolucionaron las ideas antes de convertirse en documentación oficial.

Registrar ese proceso facilita la trazabilidad, el aprendizaje y la continuidad del proyecto a largo plazo.

---

> *"A documented conversation is not a transcript. It is a record of how ideas evolved."*
